## 📌 Task 1: Iris Flower Classification

### 📖 Description
This project classifies iris flowers into three species (Setosa, Versicolor, Virginica) using machine learning.

### 🛠️ Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

### 🤖 Algorithm
- Logistic Regression

### 📊 Result
- Accuracy: 97%
- Confusion Matrix:
  [[10 0 0]
   [0 9 1]
   [0 0 10]]
